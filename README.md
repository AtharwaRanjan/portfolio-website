## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/yourusername/my-portfolio.git
```

Navigate to the project folder:

```bash
cd my-portfolio
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```
